## 📌 Overview

**Groq Data Assistant** is an AI-powered data analysis application that enables users to interact with CSV datasets using natural language. Built with LangChain and Groq, it allows you to generate insights, perform data cleaning, and create visualizations effortlessly—without writing complex code.

---

## ✨ Features

### 📂 Data Upload & Preview

* Upload CSV files instantly
* Preview structured data
* Supports real-world datasets

### 💬 Chat with Your Data

* Ask questions in natural language
* Get instant, context-aware answers
* Perform analysis without coding

### 🧹 Data Cleaning

* Identify missing values
* Detect inconsistencies
* Suggest cleaning strategies

### 📊 AI-Powered Visualizations

* Generate charts using simple prompts
* Supports bar, line, scatter, and more
* Built using Matplotlib & Seaborn

### ⚡ Fast AI Performance

* Powered by Groq for ultra-fast inference
* Uses Llama-3.3-70B model

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **AI Framework:** LangChain
* **LLM Provider:** Groq
* **Data Processing:** Pandas
* **Visualization:** Matplotlib, Seaborn

---

## 🧠 How It Works

1. Upload your CSV dataset
2. Data is loaded into a Pandas DataFrame
3. LangChain creates an intelligent agent
4. User query is processed by Groq LLM
5. Agent generates Python code dynamically
6. Code executes on your dataset
7. Results and visualizations are displayed

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/groq-ai-assistant.git
cd groq-ai-assistant
````

---

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

### 3️⃣ Activate Environment

#### Windows:

```bash
venv\Scripts\activate
```

#### Mac/Linux:

```bash
source venv/bin/activate
```

---

### 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Add API Key

Get your API key from 👉 [https://console.groq.com/](https://console.groq.com/)

Then either:

### Option 1: Streamlit Sidebar (recommended)

Paste directly in app UI

### Option 2: Environment Variable

```bash
export GROQ_API_KEY="your_key"
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```
